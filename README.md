🏦 Loan Approval Prediction System
📌 Project Overview

This project is a Loan Approval Prediction System that uses machine learning to automate the process of approving or rejecting loan applications. The model analyzes applicant details such as income, credit history, and loan amount to make accurate and data-driven decisions.

🎯 Objective
To build a predictive model that determines loan approval status
To reduce manual effort and human bias in loan decision-making
To improve accuracy using machine learning techniques
📊 Dataset

The dataset contains the following features:

Gender
Marital Status
Dependents
Education
Self Employment
Applicant Income
Coapplicant Income
Loan Amount
Loan Amount Term
Credit History
Property Area
